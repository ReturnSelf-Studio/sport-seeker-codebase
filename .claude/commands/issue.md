# /issue — Điều tra bug từ user report

Khi user report lỗi, thu thập theo thứ tự:

1. Platform (macOS / Windows)?
2. Version (`v1.x.x+xx` ở góc dưới trái app)?
3. Screenshot hoặc mô tả lỗi
4. Log nếu có (`~/SportSeeker/logs/checkpoints.log`)

Với Windows: ưu tiên check `memory/known_issues.md` trước khi điều tra sâu.
